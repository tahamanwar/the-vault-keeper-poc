# The Vault Keeper - Proof of Concept

This repository contains the Proof of Concept for "The Vault Keeper" project proposal. It demonstrates the use of a Python backend, Flask web framework, and SQLite database.

## System Specifications & Compatibility
* **Operating System:** Cross-platform (Windows 10/11, macOS, or Linux/Ubuntu 22.04+)
* **Language/Environment:** Python 3.10 or higher (Developed using Python 3.13.1)
* **Framework:** Flask 3.1.2 (To Install: `pip install flask` in the terminal)
* **Recommended IDEs:** VS Code, IntelliJ IDEA (PyCharm), or any standard text editor.

## How to Run
1. Ensure Python 3.13.1 is installed (Run: 'python --version' in the terminal)
2. Install the Flask framework (Run: 'pip install flask' in the terminal)
3. Change your directory to the project's folder and run this command: 'python app.py'
4. The terminal will show "Running on http://127.0.0.1:5000", either paste this into your web browser or ctrl + left-click the link from the terminal to open the website.
